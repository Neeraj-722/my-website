# Neeraj Kushwaha Portfolio

A high-performance, modern portfolio website built to showcase my professional experience, skills, and projects as a Software Engineer specialized in C#, .NET, Angular, and Azure.

## 🚀 Features

*   **Ultra-Modern Aesthetics**: Dark-mode primary design, glassmorphism UI features, dynamic deep-gradients, and immersive background neon animations.
*   **Fully Responsive**: Built with modern CSS Grid and Flexbox for seamless scaling on desktops, tablets, and mobile devices.
*   **Interactive UI**: Smooth cubic-bezier micro-animations, scroll-reveal mechanics, and a customized typewriter effect.
*   **Zero Dependencies**: Written purely in Vanilla HTML5, CSS3, and JavaScript—insuring lightning-fast load times.

## 🛠️ Built With

*   HTML5
*   CSS3 (Leveraging CSS Variables and Grid)
*   Vanilla JavaScript (ES6+)
*   Google Fonts: *Inter* and *Outfit*
*   FontAwesome (for Icons)

## 💻 How to Run

Since this project has zero external frameworks or Node.js dependencies:
1. Clone or download this repository to your local machine.
2. Navigate to the project folder.
3. Simply double-click `index.html` to open it in your preferred web browser.
